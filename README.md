\# FinSight AI – Intelligent Personal Finance Manager



FinSight AI is an AI-powered personal finance intelligence system that analyzes user spending, detects anomalies, forecasts future expenses, and provides insights through a conversational interface.



This project is built to demonstrate real-world application of:

\- Machine Learning

\- NLP

\- Time Series Forecasting

\- Backend Engineering

\- API Design

\- MLOps

\- LLM Integration



\## Features



\- Smart expense categorization using NLP

\- Spending pattern analysis and clustering

\- Anomaly detection for unusual transactions

\- Budget and expense forecasting

\- LLM-powered finance chatbot

\- Interactive dashboard

\- Secure REST API with JWT authentication

\- Dockerized deployment



\## Tech Stack



Backend: FastAPI, Python  

Machine Learning: Scikit-learn, PyTorch  

NLP \& LLM: OpenAI / Local LLM (Ollama)  

Database: PostgreSQL / SQLite  

Frontend: Streamlit / React  

DevOps: Docker, GitHub Actions  



\## Architecture



Frontend → FastAPI Backend → AI Engine → Database  

LLM Layer for conversational analytics
